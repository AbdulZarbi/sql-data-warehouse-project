Welocme to the **Data Warehouse and Analytics Project** repository! 🦋
This project demonstrate a comperhensive data warehousing and analytics solution, from buidling a data warehouse to generating actionable insights . Design as a portofolio project highlights industry practicse in data engineering and analytics. 

---
## Project Reqquirments 

### Building the Data Warehouse (Data Engineering) 
Develop a mdodren data warehousing using MYSQL to consolidate sales data, enabling analytics reporting and informed decision makking. 

### Specification 
- **Data Source**: Inmoort data from two source systems (ERP and CRM) provided as CSV files
- **Data Quality**: Cleans and resolve data quality issues prior to analysis.
- **Integration**: Combine both sources in a single, user-friednly data model designed for analytical queries.
- **Scope**: Focus on the latest dataset only; historirzation of data is not required.
- **Documentation**: Provide clear documnetation of the data model to support both business stakeholders and analytics team.
